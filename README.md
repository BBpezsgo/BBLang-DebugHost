# Debugger host for my [programing language](https://github.com/BBpezsgo/Interpreter)

[![.Net 9.0](https://img.shields.io/badge/.NET-9.0-5C2D91?style=flat-square)](#)
[![C# 11](https://img.shields.io/badge/C%23-11-239120.svg?style=flat-square)](#)

## Features:
- breakpoints
- variables
- expression evaluation
- bytecode inspection (it's called "disassembly" in DAP)
- step-in, -forward, -up
- memory inspection
- compiled scripts  (it's called "loaded sources" in DAP)
